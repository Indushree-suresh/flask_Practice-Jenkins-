Jenkins CI/CD Pipeline for Flask Application

Prerequisites

Before setting up the pipeline, ensure:

Jenkins installed (local VM or cloud)
Python (>=3.8) installed
pip installed
Git installed
Jenkins installed:
Pipeline Plugin

Jenkins is exposes and exposed at port 8080
